# Excel Dashboard Project

This project demonstrates data analysis and reporting using Microsoft Excel.

## Project Structure

The Excel file contains three main sheets:

### 1. Raw Data
Contains the original dataset used for analysis.

### 2. Pivot Tables
Pivot tables were created to summarize and analyze the dataset.

### 3. Dashboard
A visual dashboard built using charts and pivot tables to present key insights.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Data Analysis
- Dashboard Design

## Skills Demonstrated

- Data organization
- Data summarization
- Pivot table analysis
- Dashboard creation
- Visual reporting
