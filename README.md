# Excel Learning & Data Analysis Practices

This repository contains hands-on Excel exercises created to improve my data analysis and reporting skills.

## Content
The repository includes examples related to:

- Logical functions (IF, AND, OR, multi-criteria calculations)
- Conditional formatting (formula-based rules, icon sets, color rules)
- Lookup and reference functions
- Date and time functions
- Data cleaning and conditional calculations
- Charts and data visualization practices

## Purpose
These exercises are designed to simulate real-world business scenarios and demonstrate my ability to work with data, perform analysis, and create meaningful reports.

## Tools Used
- Microsoft Excel

## Note
This repository is for learning and practice purposes. The files reflect my progress and practical Excel skills.
